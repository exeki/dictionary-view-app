# dictionary-view-app

## Приложение для просмотра системных справочников

### Описание

Под системными справочниками в данном случае подразумеваются те справочники, список которых можно просматривать только внутри интерфейса технолога. Но и там этот список на самом деле оставляет желать лучшего, среди проблем, такие как:
1. Отсутствие фильтрации
2. Если список слишком большой, то не все объекты отрисовываются, и их не найти даже при помощи ctrl + f
3. Отсутствие пагинации
4. Его не могут просматривать и изменять обычные пользователи без доступа в интерфейс технолога
5. Отсутствие экспорта
6. Он уродливый
И вот, ради решения это проблемы было сделано это приложение (но по правде от скуки).
Перед вами встроенное приложение Naumen Service Desk, которое призвано отображать системные справочники SD красивенько и с различными дополнительными возможностями.

### Фичи

Все фичи отталкиваются от минусов коробочного списка:
1. Фильтрация, много фильтрации
2. Фильтрация - значит поиск
3. Настраиваемая пагинация 
4. Доступ вы можете выдать любым группам пользователей
5. Экспорт списка в наличии
6. Целых два режима отображения: плоский список и иерархия - и на обоих работает фильтрация и сортировка
7. Красивенько 

### Установка

3. Загрузите
